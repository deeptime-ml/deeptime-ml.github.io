function implied_timescales¶

deeptime.util.validation.implied_timescales(models, n_its=None)¶

Converts a list of models to a ImpliedTimescales object.

(Source code, png, hires.png, pdf)

Parameters

  • models (list) – The input data. Models with and without samples to compute confidences should not be mixed.

  • n_its (int or None, optional) – Number of timescales to compute.

Returns

its_data – The data object.

Return type

ImpliedTimescales
