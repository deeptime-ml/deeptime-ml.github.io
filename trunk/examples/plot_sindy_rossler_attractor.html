Identification of the Rossler system with SINDy¶

This example shows how to use SINDy to discover the chaotic Rossler system from measurement data via the deeptime.sindy.SINDy estimator and deeptime.sindy.SINDyModel model. Once we’ve learned the system, we can also simulate forward in time from novel initial conditions.

Note that for this example we pass in the exact derivatives. In practice one can also pass in a numerical approximation in their place.
