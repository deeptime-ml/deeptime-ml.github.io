class LaggedModelValidator¶

class deeptime.util.LaggedModelValidator(test_model: deeptime.base.Model, test_estimator: deeptime.base.Estimator, test_model_lagtime: int, mlags, conf=0.95, err_est=False)¶

Validates a model estimated at lag time tau by testing its predictions for longer lag times

Parameters

  • test_model (Model) – Model to be tested

  • test_estimator (Estimator) – Parametrized Estimator that has produced the model

  • mlags (int or int-array, default=10) – multiples of lag times for testing the Model, e.g. range(10). A single int will trigger a range, i.e. mlags=10 maps to mlags=range(1, 10). The setting None will choose mlags automatically according to the longest available trajectory

  • conf (float, default = 0.95) – confidence interval for errors

  • err_est (bool, default=False) – if the Estimator is capable of error calculation, will compute errors for each tau estimate. This option can be computationally expensive.
