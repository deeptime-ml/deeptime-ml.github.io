class deeptime.decomposition.DMDModel(eigenvalues: numpy.ndarray, modes: numpy.ndarray, mode='exact')¶

  • modes ((n, n) ndarray) – The DMD modes.

  • mode (str, optional, default='exact') – The mode of estimation that was used. See DMD.available_modes.

Set the parameters of this estimator.
